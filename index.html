プライバシーポリシー

星街すいせい個人事務所 Studio STELLAR（以下「当事務所」といいます。）は、個人情報の重要性を認識し、本ポリシーに従って適切に管理いたします。

1. 個人情報の範囲
当事務所は、次の各号に該当する情報のうち、利用者個人を識別することができる情報を個人情報として扱います。
当事務所サービスの会員登録画面、商品購入申込画面、その他の画面等において、利用者が入力し当事務所に提供した情報
当事務所サービスに関連して、利用者から提供を受けたその他の情報
なお、当事務所では、事前に利用者に通知した上で当事務所が収集業務を委託する会社（当事務所と守秘義務契約を締結します）を除き、当事務所以外の第三者が個人情報を収集することはありません。
また、当事務所サービスのリンク先など、当事務所以外の第三者が個人情報を収集するウェブサイトについては、本プライバシーポリシーは適用されません。当事務所では一切の義務や責任を負いかねますので、各サイトのプライバシーポリシーをご確認ください。
なお、当事務所サービスにおいて公開される情報は、利用者個人が識別されない範囲に限ります。

2. 情報の収集
当事務所では、当事務所サービスをご利用いただくために必要な情報のほか、利用者の皆様により便利にサービスをご利用いただくことを目的として個人情報を収集する場合があります。
また、以下のような場合に情報を取得することがあります。
キャラクターのなりきり機能のため、Appleが提供する TrueDepth API などを利用して顔の表情データを取得する場合
取得したデータはキャラクター機能のためだけに使用します
データを保存したり第三者に共有することはありません
当事務所または提携企業が提供する懸賞企画やアンケートページ、イベント等で個人情報をお聞きする場合
会員が当事務所サービスにログインした場合、当該会員のサービス利用状況が当事務所で把握できる場合
利用者が当事務所サービスのページを閲覧・利用する際に、IPアドレスやクッキー情報などを自動的に取得します。

3. 情報の利用および開示
当事務所は、収集した個人情報を以下の目的で利用します。
利用者の利用状況や趣味嗜好に適合したサービスや広告を提供・表示するため
利用者の興味関心を分析し、新規サービスを開発するため
プレゼント企画等で当選した場合に商品を配送するため
サイト運営上必要な告知を行うため
統計分析のため
なお、統計分析結果は第三者に公開する場合がありますが、個人を特定できる情報は含まれません。

個人情報の第三者提供
当事務所は、原則として利用者の個人情報を第三者に開示することはありません。ただし、以下の場合を除きます。
利用者本人の同意がある場合
サービス提供のため、当事務所から委託を受けた会社（守秘義務契約締結済み）が業務に必要とする場合
例：配送業者への住所情報の提供など
裁判所、弁護士会、警察等の公的機関から法令に基づく照会を受けた場合
当事務所の権利、財産、サービスを保護する必要がある場合
人命、身体、財産などに差し迫った危険があり、緊急の必要性がある場合
また、提携企業がイベントなどで利用者の情報を収集・利用する場合には、都度その旨を利用者に明示し、本人の同意を得た上で情報を提供します。

4. クッキー（Cookie）について
当事務所サービスの一部では、より快適なサービス提供のため クッキー（Cookie） を利用しています。
クッキーとは、ウェブサイト閲覧時にブラウザとサーバ間で送受信される小さなファイルであり、利用者のコンピュータに保存されます。
クッキーを使用することで、当事務所サービスのサーバは以下の情報を記録できます。
どのページを訪問したか
利用状況などの情報（クッキー情報）
ただし、利用者が個人情報を入力しない限り、個人を特定することはありません。
利用者はブラウザ設定により以下を選択できます。
クッキーの受取を拒否する
クッキー受信時に警告を表示する
ただし、クッキーを拒否した場合、当事務所サービスの一部が利用できなくなる場合があります。

クッキーの利用目的
当事務所では、クッキーを以下の目的で利用します。
不正利用の確認
会員確認および登録情報へのアクセス
効果的な広告表示
利用者数の計測および統計分析

5. アクセスログについて
当事務所は、利用者が当事務所サービスのウェブページにアクセスした際、その情報を アクセスログ としてサーバに記録し一定期間保存します。
アクセスログには以下の情報が含まれます。
アクセス日時
IPアドレス
使用ブラウザの種類
クッキー情報
これらの情報は、以下の目的で利用されます。
サーバ運用の改善
サーバ管理
不正利用の調査

6. 登録情報に関するお問い合わせ
会員登録時に登録された個人情報は、当事務所サービス内の所定ページにて随時変更することができます。
なお、以下の場合は当事務所までお問い合わせください。
ご自身で情報変更ができない場合
ご自身の個人情報に関する確認・問い合わせを行いたい場合

7. 規約の改訂
当事務所は、各種規約・ガイドライン・約款（以下「規約等」といいます。）を、会員への事前通知なしに改訂することがあります。
改訂された規約等は、改訂前後を問わずすべての会員に適用されます。改訂内容は、サイト上または電子メールにより告知されます。
